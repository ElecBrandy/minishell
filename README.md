## 05.15
### 1. 의문의 엔터 (완)
- 미니쉘 진입 후 예를 들어 `cat a.txt`시 결과가 나온 히후 다시 nimishell$ 문구와 함께 명령어를 입력하려면 엔터를 한번 더 쳐야 진입이 가능했던 것 
- 파이프가 없을 경우 자식 프로세서를 기다리지않고 부모프로세서가 먼저 시작해서 그런 경우 -> `wait_process`를 통해서 자식 프로세서가 끝나는 것을 기다리고 실행되게 끔 수정
